Welcome to Agni’s documentation!¶

About¶

Agni is a theme for the Python Sphinx documentation system.

How to use¶

  1. Download agni-1.0.zip and uncompress the archive.
  2. Modify the conf.py of an existing Sphinx project or create a new Sphinx project using sphinx-quickstart.
  3. Change the html_theme parameter to agni.
  4. Change the html_theme_path to location of the downloaded theme.

Credits¶
